## REPOSITORIO DE EJEMPLOS BÁSICOS PARA ARDUINO
------

En cada directorio encontrarás:

- Código fuente (.INO)
- Binario Arduino (.HEX)
- Librerías requeridas (.ZIP)
- Circuito simulable mediante SimulIDE (.SIMU)
- Diagrama de conexiones (.PNG)
- Eagle PCB (.BRD) 
- Hoja de datos (.PDF)
